						Chessboard
======================================================================================================
The application calculates a list of all possible paths that one knight piece in the starting position
could take to reach the ending position in 3 moves.
======================================================================================================
* Created by: Andri Eracleous
* Date:       29/01/2022
======================================================================================================
* Algorithm: Breath First Search (BFS)
======================================================================================================
* Input: Command Line
  Example:
               Welcome!
----------------------------------
8|   |   |   |   |   |   |   |   |
----------------------------------
7|   |   |   |   |   |   |   |   |
----------------------------------
6|   |   |   |   |   |   |   |   |
----------------------------------
5|   |   |   |   |   |   |   |   |
----------------------------------
4|   |   |   |   |   |   |   |   |
----------------------------------
3|   |   |   |   |   |   |   |   |
----------------------------------
2|   |   |   |   |   |   |   |   |
----------------------------------
1|   |   |   |   |   |   |   |   |
----------------------------------
   A   B   C   D   E   F   G   H 

Enter the Origin Point: A1
Enter the Destination Point: B4
======================================================================================================
* Output: The path will appear as below
B4
C2
A1
